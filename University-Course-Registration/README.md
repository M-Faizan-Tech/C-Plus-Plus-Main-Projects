# 🏫 University Course Registration & Recommendation System

An academic routing console application designed to display course hierarchies, prerequisite requirements, and credit hours mapping dynamically across multiple departments.

## 🧠 Architectural Logic
- **Conditional Mapping Trees:** Implements deep nested conditional flows (`if-else` blocks) to filter curriculum data based on user input parameters.
- **Academic Roadmap Validation:** Automatically prints degree prerequisites and maps student journeys for CS, EE, ME, and BBA paths.

## 🛠️ Concepts Covered
- Nested Decision Logic Mapping
- Academic Prerequisite Layouts
- Multi-string Input Filtering
